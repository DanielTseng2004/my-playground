# PlayGround

## Introduction

A Vue 3 + TypeScript + Vite Project that includes multiple games, such as 2048, Texas Hold'em, Blackjack, TTT, Minesweeper, Wordle, Number Puzzle, and Slots. The project is designed to provide a fun and interactive gaming experience while showcasing the capabilities of Vue 3 and TypeScript.
Created by ManusAI

### Project setup & Run

```bash
npm install
npm run dev
```
